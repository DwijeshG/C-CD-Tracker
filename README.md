# C-CD-Tracker
A modular C++ CLI application designed to manage a personal music CD collection with integrated utility tools.
Collection Management: Supports reading data from file collections (cd.txt), displaying detailed inventories, and sorting records alphabetically by artist using an optimized Selection Sort.

Search Mechanics: Implements both iterative sequential scanning and recursive lookups to find specific albums.

Dynamic Track Allocation: Features custom runtime track assignment leveraging dynamic arrays, standard vectors, and deep-copy semantics (copy constructors/destructors) for safe memory handling.

In-App Utility Modality: Includes an interactive "Math Break" sub-menu supporting custom parity validation, basic prime factoring optimization, and recursive string reversal algorithms.
